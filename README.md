# Tassku Website

Marketing website for [Tassku](https://github.com/alvaro347/Tassku), a modern task manager built with Kotlin Multiplatform.

## Tech Stack

- React + TypeScript
- Vite
- CSS Modules with custom properties
- Lucide React icons

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Automatically deployed to GitHub Pages on push to `main` via GitHub Actions.
